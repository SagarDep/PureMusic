## Pure Music Player
#### PureMusic是一款开源Android在线音乐播放器。

* 播放本地音乐与在线音乐
* 在线音乐播放等
* 高仿云音乐的黑胶唱片专辑封面
* 歌词显示，自动搜索歌词
* 编辑歌曲信息
* 定时关闭
